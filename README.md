# Order Price Calculator (JPY → VND)

Vite + React + Tailwind app. Ready for Netlify (CI/CD).

## Development
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to Netlify (Method 3)
- Connect repo in Netlify → Build command: `npm run build`, Publish dir: `dist`
- Or use `netlify.toml` already included.
